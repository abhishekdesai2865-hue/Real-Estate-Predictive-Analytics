# Real Estate Predictive Analytics Project

## Overview
This project analyzes the financial performance of leading Indian real estate companies and builds machine learning models to predict residential property prices.

Companies analyzed:
- Lodha Group
- Godrej Properties
- Prestige Group
- DLF Limited

The project integrates financial analysis, predictive modeling, and scenario forecasting to support data-driven investment and business decisions.

## Objectives
- Analyze revenue, EBITDA, PAT, CAGR, and leverage trends of listed developers.
- Study pricing behavior using property-level data.
- Build predictive models to forecast property prices.
- Compare Linear Regression, Random Forest, and XGBoost models.
- Perform scenario forecasting (Slow, Baseline, Strong).

## Dataset
- 500 residential properties
- Features include city, carpet area, bedrooms, amenities, furnishing type, developer brand, and location type.

## Tools & Technologies
- Excel (Data Cleaning & Financial Analysis)
- Power BI (Dashboards & Visualization)
- Python (Pandas, Scikit-learn, XGBoost)
- Machine Learning Models

## Key Results
- XGBoost achieved high prediction accuracy (R² ≈ 0.92).
- Carpet area, city, developer brand, and amenities are major price drivers.
- Scenario analysis shows different price growth patterns under economic conditions.
